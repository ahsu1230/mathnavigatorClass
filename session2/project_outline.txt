-------------------------------
Project Outline
Session 2
Centering a header and content
-------------------------------

Create a new Project...

See if you can create a separate "header" and "content" section.
Use colors to differentiate the two sections.

Have the "header" section be the width of the window and be narrow in height
(i.e. 60px).

Have the "content" section is one of the following widths:
480px
560px
640px
720px
800px

And then center it horizontally using CSS { margin: 0 auto; }
